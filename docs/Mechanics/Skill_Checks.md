# Mechanics: Skill Checks

## 1. Skill List
Checks typically use the formula `d20 + Ability + Prof`.

| Skill | Ability | Description |
| :--- | :--- | :--- |
| **Acrobatics** | DEX | Balance, flips, escaping grapples. |
| **Animal Handling** | WIS | Calming beasts, mounting, controlling mounts. |
| **Arcana** | INT | Knowledge of magic, planes, symbols. |
| **Athletics** | STR | Climbing, jumping, swimming, grappling. |
| **Deception** | CHA | Lying, conning, misleading. |
| **History** | INT | Knowledge of past events, lore per se. |
| **Insight** | WIS | Detecting lies, reading emotions. |
| **Intimidation** | CHA | Threats, coercion. |
| **Investigation** | INT | Searching for clues, deducing info. |
| **Medicine** | WIS | Stabilizing dying, diagnosing illness. |
| **Nature** | INT | Knowledge of flora, fauna, weather. |
| **Perception** | WIS | Spotting hidden enemies, hearing noises. |
| **Performance** | CHA | Music, acting, storytelling. |
| **Persuasion** | CHA | Diplomacy, negotiation. |
| **Religion** | INT | Knowledge of deities, cults, rites. |
| **Sleight of Hand** | DEX | Pickpocketing, concealing objects. |
| **Stealth** | DEX | Hiding, moving silently. |
| **Survival** | WIS | Tracking, foraging, navigating. |

## 2. Difficulty Class (DC) Table
The Engine sets the DC based on the task difficulty.

| Difficulty | DC | Example |
| :--- | :--- | :--- |
| **Very Easy** | 5 | Climbing a knotted rope. |
| **Easy** | 10 | Hearing a guard approach in armor. |
| **Medium** | 15 | Pickpocketing an alert merchant. |
| **Hard** | 20 | Swimming in a storm; tracking on stone. |
| **Very Hard** | 25 | Disarming a masterwork trap. |
| **Nearly Impossible** | 30 | Tracking a ghost across water. |

## 3. Passive Checks
Used when the character is not actively rolling but the environment acts upon them.
*   **Formula:** `10 + Modifier` (e.g., Passive Perception = 10 + WIS Mod + Prof).
*   **Advantage/Disadvantage:** +/- 5 to the Passive Score.
